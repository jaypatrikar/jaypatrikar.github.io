## Font Combination Options

### Option A: Inter + Playfair Display (Serif Headings)
**Current**: Inter for everything
**New**: 
- Headings: Playfair Display (elegant serif)
- Body: Inter (clean sans-serif)
- **Vibe**: Sophisticated, readable, classic academic

### Option B: Inter + Space Grotesk
**Current**: Inter for everything
**New**:
- Headings: Space Grotesk (geometric, modern)
- Body: Inter (clean sans-serif)
- **Vibe**: Modern, tech-forward, geometric

### Option C: Source Sans Pro + Crimson Text
**Inspired by**: Stanford GSE
- Headings: Source Sans Pro
- Body: Crimson Text (serif)
- **Vibe**: Traditional academic, readable

### Option D: Inter + JetBrains Mono (Code-friendly)
**For tech/robotics focus**:
- Headings: Inter
- Body: Inter
- Code/Technical: JetBrains Mono
- **Vibe**: Developer-friendly, technical

### Option E: Inter + Literata (Serif Body)
**Modern academic**:
- Headings: Inter
- Body: Literata (serif, optimized for reading)
- **Vibe**: Best readability, scholarly

### Option F: Keep Inter (Current)
**Already excellent** - just refine weights and spacing
